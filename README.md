# Print Sum
## Problem Descrition ##
輸入2整數 $a$, $b$, 請印出其之和

## Input Format ##
共有一行輸入包含二整數 $a$,$b$
## Constraints ##
$0 < a,b < 100000$
## Output Format ##

共有一行輸出，印出 $a+b$ 的值

## Sample Input 1 ##
```
4
2
```
## Sample Output 1 ##
```
6
```
## Sample Input 2 ##
```
48
32
```
## Sample Output 2 ##
```
80
```
